# Python Millionaire Game

A console-based quiz game inspired by "Who Wants to Be a Millionaire".

## Features

- 10 multiple-choice questions
- User answer validation
- Instant feedback
- Game over on incorrect answer

## Technologies Used

- Python

## How to Run

```bash
python main.py
```
